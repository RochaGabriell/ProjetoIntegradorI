Guia básico de Markdown

Markdown é uma linguagem de marcação popular e intuitiva que permite a formatação fácil de textos para a web. Com ela, é possível criar diversos elementos como títulos, listas, links, imagens e muito mais.

Confira abaixo um guia com todas as funcionalidades do Markdown disponíveis na plataforma.

Títulos

Para criar títulos, adicione um ou mais sinais de hashtag (#) antes do texto do título. Quanto mais hashtags, menor o título.

Exemplo:

# Título de nível 1
## Título de nível 2
### Título de nível 3

Texto em negrito e itálico

Para deixar o texto em negrito, envolva-o com dois asteriscos (**). Para deixar o texto em itálico, envolva-o com um asterisco (*).

Exemplo:

**texto em negrito**
*texto em itálico*

Listas

Para criar uma lista não ordenada, use um asterisco (*) ou um sinal de menos (-) antes de cada item. Para criar uma lista ordenada, use um número seguido de um ponto (.).

Exemplo:

* Item 1
* Item 2
    * Item 2a
    * Item 2b
1. Item 1
2. Item 2

Links

Para criar um link, coloque o texto do link entre colchetes [] e a URL entre parênteses ().

Exemplo:

[Google](https://www.google.com/)

Imagens

Para inserir uma imagem, use a mesma sintaxe de um link, mas adicione um ponto de exclamação (!) antes dos colchetes.

Exemplo:

![Pato fofo](https://www.example.com/pato.jpg)

Parágrafos

Para criar um novo parágrafo, basta deixar uma linha em branco entre eles.

Exemplo:

Este é um parágrafo.

Este é outro parágrafo.

Quebras de linha

Para criar uma quebra de linha sem criar um novo parágrafo, use dois espaços no final da linha.

Exemplo:

Esta linha  
será quebrada em duas.

Código

Para formatar um trecho de código, envolva-o com um acento grave (`). Para formatar um bloco de código, envolva-o com três acentos graves (```).

Exemplo:

```    
public class Simples {
    public static void main(String[] args) {
            System.out.print(“Gabriel”);
            System.out.print(“Silva Moraes”); 
    }
}
```
