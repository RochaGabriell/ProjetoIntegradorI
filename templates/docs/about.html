Sobre Dev.Duck

Trata-se do desenvolvimento de um sistema web, com o objetivo de criar uma plataforma simples onde qualquer estudante pode publicar uma vez ou ocasionalmente, que também permite ao escritor compartilhar seus conhecimentos ou experiências sobre determinada matéria dentro de sua grade curricular.

É um software de código aberto projetado para ajudar a comunidades academicos do IFPI - Campus Corrente. Como nosso site é de código aberto, você pode inspecionar cada pequeno detalhe do código e inserir você mesmo uma nova funcionalidade!
